Painterly 鎏金 bitmap button for hero / feature CTAs. Set `window.LF_ICON_BASE` once per page. For dense UI use the crisp `Button`.

```jsx
<OrnateButton size="lg" icon={<i data-lucide="wand-sparkles" />}>开始生成</OrnateButton>
<OrnateButton variant="paper">暂不领取</OrnateButton>
```

`variant`: teal | paper. Stretches the plate art to fit — keep widths moderate so the ornate ends don't over-distort.
